Hyphens? Not a problem!

Your next task is to write a program that will match hyphenated words, such as:

  • twenty-one
  • long-term
  • co-worker
  • well-known

It should return True if the word is written with a dash, and False otherwise.


Sample Input:
good-looking

Sample Output:
True
