Picky parents

A couple of friends that you know have recently become parents. Moreover, they have bought a puppy! However, they are having a hard time coming up with a name for it. However, they have decide that the name to start with any letter from B to N and the second character has to be a vowel.

Write a program that checks whether the name suits these two criteria and prints "Suitable!" if a match is found.


Sample Input:
Butterfly

Sample Output:
Suitable!



Memory limit: 256 MB
Time limit: 15 seconds
