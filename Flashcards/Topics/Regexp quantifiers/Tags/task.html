Tags

Apply your skills to write a regular expression searching for a pair of tags.

  • the first tag from the pair looks like <...> where ... stands for a non-empty string of any length composed of any lowercase Latin letters
  • the second tag from the pair looks like </...> where ... stands for a non-empty string of any length composed of any lowercase Latin letters
  • between those tags, there can be any number of any symbols
  • the ... part of these tags don't have to be identical (that is, the names of the tags can be different)

Note that your template should match only substrings that contain one pair of such tags.

Put your regexp in the variable template.
